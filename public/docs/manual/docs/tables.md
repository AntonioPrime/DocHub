# Таблицы (Table)

Предназначен для отображеия данных в табличной форме.

# Предопределенный набор данных
```yaml
docs:
  dochub.table.preset:
    data:
      - id: 1
        payload: Строка 1
      - id: 2
        payload: Строка 2
```
